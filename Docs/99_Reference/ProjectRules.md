## 概要

## 技術スタック

-   Godot Engine 4.x
 -   C# (Godot .NET)
-   GitHub + Cursor

## コーディングルール

-   コメントは日本語で書くこと
-   再利用性と可読性を重視
-   ファイル名・ノード名は一貫性を保つ
-   大きな機能は分割して実装
-   理由と目的を明確に記述
