## Issue テンプレート

各 Issue には以下の情報を含めてください：

```markdown
## 説明

[具体的な実装内容]

## 受け入れ基準

-   [ ] 基準 1
-   [ ] 基準 2
-   [ ] 基準 3

## 技術的な注意点

-   実装時の考慮事項
-   パフォーマンス要件
-   テスト要件

## 関連 Issue

-   依存する Issue
-   関連する Issue

## 見積もり

-   工数: X 時間
-   優先度: High/Medium/Low
```
